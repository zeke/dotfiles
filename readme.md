# zeke/dotfiles

## Setup

1. Install [Homebrew](https://brew.sh)
2. Run the setup script:

```sh
./script/setup
brew bundle
```

## Terminal

- [pure](https://github.com/sindresorhus/pure) prompt (installed via Brewfile)
- [Snazzy](https://github.com/sindresorhus/terminal-snazzy) theme for Terminal.app
